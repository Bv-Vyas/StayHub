
# StayHub - Full Stack Project
#Overview

StayHub is a full-stack web app for travelers to explore and book hotels, properties, and rental lands. It uses HTML, CSS, JavaScript, Node.js, Express, EJS, and MongoDB.




## Authors

- [@Bv-Vyas](https://github.com/Bv-Vyas)


## Features

- Upload Listings
- Reviews and Ratings
- Authentication
- Validation
- Third-Party Integration (MongoDB Atlas, Cloudinary)


## Tech Stack

**Client:** HTML, CSS, JavaScript, EJS

**Server:** Node.js, Express

**Database:** MongoDB (MongoDB Atlas)


## Getting Started

- Clone repository: git clone <https://github.com/Bv-Vyas/StayHub>
- Install dependencies: npm install
- Set up MongoDB Atlas and Cloudinary.
- Configure environment variables.
- Run: npm start
